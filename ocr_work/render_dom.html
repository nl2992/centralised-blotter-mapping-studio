Mapping Studio
Precedence: manual override > user rule (asset-class rule wins over Global) > reference lookup > built-in default > placeholder. Drag a source field onto a template field to wire it (solid red line = your rule, dashed gray = built-in default; click a red line or the ✕ to remove a rule). After wiring, set a conversion in the Configuration column (scale ×, MM→USD, date→ISO, UPPER/lower, value-map like New=>Buy; Unwind=>Sell, or a custom expression using value) to transform the source on top of the default.
